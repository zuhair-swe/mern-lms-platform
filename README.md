# 🎓 MERN LMS Platform

A full-stack Learning Management System (LMS) built using the MERN stack.  
This platform enables admins, teachers, and students to manage courses, users, and learning activities efficiently.

---

## 🚀 Features

### 🔐 Authentication & Authorization
- Role-based login (Admin, Teacher, Student)
- JWT-based authentication
- Secure route protection (middleware)

---

### 👨‍💼 Admin Panel
- Approve or reject teachers
- View all teachers and students
- Delete/manage teacher accounts
- Dashboard with pagination

---

### 👩‍🏫 Teacher Features
- Add new students
- Manage student data
- Send login credentials via email

---

### 👨‍🎓 Student Features
- Login with email credentials
- View profile
- Access learning system

---

### 📧 Email System
- Automated email sending using Nodemailer
- Sends login credentials to students

---

### 🖼️ File Uploads
- Profile image upload using Multer

---

### ⚙️ Other Features
- Pagination in dashboards
- Form validation
- Clean UI & structured backend

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Authentication
- JSON Web Token (JWT)
